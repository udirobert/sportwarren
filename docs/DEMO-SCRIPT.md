# Demo Script (2 minutes)

> Record on your phone showing WhatsApp. Screen-record the Basescan verification on desktop. Splice together.

---

## [0:00–0:15] Hook

"SportWarren gives every amateur football team an autonomous AI agent — a Squad Manager Twin — that scouts opponents, pays for intelligence from the Kite marketplace, and manages its own budget. Players just text WhatsApp. No wallets, no apps, no crypto knowledge."

## [0:15–0:40] Show kite-proof (on-chain payment)

*[Show WhatsApp, type `kite-proof`, wait for response with Basescan link]*

"Here's the agent paying for a service in real-time. I text 'kite-proof' — the agent discovers a Kite-listed weather API, negotiates an x402 payment, and settles in USDC. That Basescan link is a real on-chain transaction — anyone can verify it."

*[Tap the link or cut to browser showing Basescan tx]*

"One dollar cent, settled on Base through Kite Passport. No human approved this specific payment — the agent operates within a pre-approved spending session with hard limits."

## [0:40–1:15] Show scout (autonomous intelligence)

*[Back to WhatsApp, type `scout Chelsea`]*

"Now the core use case. A player texts 'scout Chelsea' — the agent checks their identity, verifies budget, generates tactical intelligence, and records an attestation. All autonomous."

*[Wait for scout response to appear]*

"Formation, strengths, weaknesses, and a counter-strategy — tailored for an amateur squad. The agent spent from its own budget, enforced daily caps, and persisted a receipt."

## [1:15–1:40] Show autonomy + constraints

*[Type `budget`]*

"Every agent has programmable spending constraints. Per-transaction limits, daily caps, per-squad isolation. The agent can't overspend even if you ask it to."

*[Brief pause on budget response]*

"And this isn't just on-demand. A cron job fires 22 hours before every match — the agent scouts the opponent automatically and delivers the report to the squad's WhatsApp group. Zero human trigger."

## [1:40–2:00] Wrap-up

"SportWarren is live in production right now. Real Kite Passport agent identity, real x402 payments, real spending sessions with passkey approval. Two billion WhatsApp users can access autonomous AI commerce without ever touching a wallet.

Text plus-one-two-oh-one-five-three-four-five-three-eight-four and try it yourself. Thank you."

---

## Tips

- Keep energy up, don't read robotically
- Have WhatsApp open and ready before recording — responses take 8-12 seconds
- Show the Basescan page loading with the real tx amount visible
- If scout takes too long, cut and resume (judges won't mind a clean edit)
- End on the WhatsApp number so judges can try it themselves
